## üìñ About
This project is a pixel-perfect clone of the [Cursor](https://veeshal.me/chaicode/cursor) landing page. It is built using **semantic HTML and Vanilla CSS** to replicate the sleek, dark-themed aesthetic of the original site. The project focuses on complex grid layouts, responsive components, and fine-tuned typography without utilizing any external CSS frameworks.

---

## üé® Features
- **AI-Focused UI**: Deep dark background (`#14120b`) with high-contrast text and accents.
- **Complex Layouts**:
  - CSS Grid for the "Trusted By" and "Changelog" sections.
  - Alternating feature sections (Text/Image split).
  - Bento-grid style testimonials.
- **Typography**: Integration of **Plus Jakarta Sans** for a modern, clean look.
- **Responsive Design**: Fluid containers and flexible flexbox/grid structures.
- **Interactive Elements**: Hover effects on buttons (`.nav-btn`, `.cta`) and cards.
